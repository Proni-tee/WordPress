How to Use WordPress

Download and Install XAMPP

  • Download XAMPP
  • During installation, select Apache, MySQL, PHP, and phpMyAdmin.
    • Launch XAMPP and start Apache and MySQL modules.

Download WordPress

  • Download WordPress
  • Unzip the WordPress files into the C:\xampp\htdocs\.

    • Create a WordPress Database

  • Open your browser and go to http://localhost/phpmyadmin
  • Click "New" on the left sidebar.
  • Enter a database name (e.g., wordpress_db) and select utf8_general_ci. Click Create..

    • Configure WordPress

  • Open your browser and go to http://localhost/wordpress

    • Fill in the database details

    • Database Name: wordpress_db.
    • Username: root.
    • Password: Leave blank
    • Database Host: localhost
  • Click Submit and follow the setup process to create an admin account
